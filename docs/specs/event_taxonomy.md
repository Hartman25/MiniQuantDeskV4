# Event Taxonomy Spec (V4)

All events use a standard envelope:
event_id, run_id, engine_id, ts_utc, correlation_id, causation_id, topic, event_type, payload.

Topics: MARKET, STRATEGY, OMS, RISK, BROKER, LEDGER, RECON, RUNTIME, AUDIT, EVAL, PROMOTION.

Key event families include:
- MARKET_BAR, MARKET_DATA_STALE/GAP/OUTLIER, DATA_SOURCE_SWITCH, DATA_DISAGREEMENT
- SIGNAL_INTENT_EMITTED, SHADOW_ORDER_INTENT, STRATEGY_ERROR
- ORDER_CREATED/SUBMITTED/FILLED/CANCELED/REPLACED/REJECTED
- OUTBOX_ENQUEUED/INBOX_RECEIVED/BROKER_FILL/BROKER_REJECT
- FILL_APPLIED/POSITION_UPDATED
- RECON_* cycle events + mismatch types
- ARMING_* / ARMED / DISARM_* / DEADMAN_TRIGGERED / LOOKAHEAD_VIOLATION
- EVAL_* / PROMOTION_*
